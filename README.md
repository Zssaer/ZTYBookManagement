# BookManagement
 该项目为Android开发实训项目。
       该图书后台管理系统APP的API基于安卓10开发，使用了DrawerLayout侧拉栏以及Litepal进行SQL操作，拥有一键添加图书数据功能以及自定义图书功能，其图书展示为Material Design卡片设计风格，含有下滑刷新功能，全活动采用全沉浸状态栏设计，功能界面采用了Fragment碎片方法。
